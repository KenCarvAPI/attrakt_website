## Overview

Attrakt helps technology companies (Web2 and Web3) build organic growth through developer advocacy. Our approach combines:

- **Premium Services** — Developer experience, community growth, and product marketing
- **Attrakt Intelligence** — Analytics platform for community health, champion identification, and program ROI
- **AI Agents** — Docs, Community, Audit, and Growth agents that multiply your team's impact
- **Physical Activations** — Founder hubs, exclusive events, and conference presence

## Tech Stack

This is a single-page static website built with:

- Vanilla HTML/CSS/JS
- Google Fonts (Playfair Display + Outfit)
- CSS animations and transitions
- Intersection Observer for scroll reveals

## Development

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/attrakt-website.git
cd attrakt-website

# Install dependencies (optional, for dev server)
npm install

# Start dev server
npm run dev

# Or just open index.html in your browser
open index.html
```

## Deployment

This is a static site that can be deployed to:

- **Vercel**: `vercel deploy`
- **Netlify**: Drag and drop the folder
- **GitHub Pages**: Enable in repo settings
- **Cloudflare Pages**: Connect repo
